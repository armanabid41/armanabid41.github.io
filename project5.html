Project Overview

I designed this premium, "app-like" spreadsheet to help users track their income , expenses , and financial goals without the hassle of manual calculations . Whether you are looking for a clean, Etsy-style budget planner or a robust data analytics tool for personal finance, this dashboard has it all.

Features

  • Smart Data Import: A backend engine that automatically categorizes raw transaction data using advanced array formulas.
  • Dynamic Dashboard: Real-time visual analytics including Donut charts for expense breakdowns and Area charts for savings trends.
  • Automated Monthly Tracking: "Write once, update everywhere" logic that auto-fills 12 monthly tabs.
  • Visual Goal Trackers: Built-in 50/30/20 budget rule calculator, Sinking Funds manager, and Debt Payoff tracker using custom Sparklines.
